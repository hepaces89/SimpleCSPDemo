This page has a CSP meta tag in its header. This header restricts the page javascript that comes from this domain. As a result when we tried to load jQuery from an external CDN we got an error. If you go to the Web Browser Console, you can see that when we attempt to load jQuery from a CDN on this page, it completely fails with the following error message: Content Security Policy: The page's settings blocked the loading of a resource at https://code.jquery.com/jquery-2.2.3.min.js ("default-src http://localhost:8080").

Also the report uri option is not supported via the meta element: Content Security Policy: Ignoring source 'report-uri' (Not supported when delivered via meta element).
